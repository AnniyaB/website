Things to do

Note the following refinement to the PR workflow

You are no longer required to add a Ready for review comment to every PR (If there is no comment, we assume the PR is ready for review).
Instead, if the PR is not ready for review yet, add a comment Work in progress. When the PR is ready, add a comment Ready for review.

Configure Eclipse indentation style

Configure Eclipse to automatically convert tabs to 4 spaces.
